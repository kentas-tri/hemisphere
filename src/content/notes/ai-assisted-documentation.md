## Context

The site rebuild needs a way to preserve intent between sessions.

## Decision

Use short Markdown notes and handoff records as the working memory layer.

## Reasoning

The useful part is not automation alone; it is a repeatable loop for preserving authorship and context.

## Implementation

The handoff file, commit history, and small notes become a memory layer for the lab.

## Result

The site can keep a visible record of decisions without turning the homepage into a journal.

## Next

Keep refining the notes workflow through Obsidian-authored Markdown files.

# Hemisphere

Private engineering lab for interfaces, automation, and media systems.

## Stack

- Astro
- React
- Tailwind CSS
- GitHub Actions
- GitHub Pages
- Codex

## Routes

- `/`
- `/projects/`
- `/systems/`
- `/notes/`
- `/archive/`
- `/contact/`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
